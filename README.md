# Dagbuilder

> Load a DAG described in a file into IPFS/IPLD.


## Table of Contents

- [Install](#install)
- [Usage](#usage)
- [Contribute](#contribute)
- [License](#license)


## Install

```sh
> git clone https://github.com/vmx/dagbuilder.git
> cd dagbuilder
```


## Usage

### Load data into IPLD

```sh
> node jsipld.js mixed2.dag
```
